# Column and Merging

## developer_journey_trackings (Activity)
### Column used
- first_opened_at, 
- completed_at, 
- last_viewed

### Column result: 
- avg_study_duration,
- consistency_ratio  

## exam_registration (Quiz)
### Column used: 
- deadline_at, 
- exam_finished_at

### Column result: 
- avg_time_utilization

## exam_result (Quiz Result)
### Column used:
- score, 
- is_passed
### Column result:
- average_score, 
- pass_rate
